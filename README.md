🎓 Alumni Tracker System


Alumni Tracker System adalah aplikasi berbasis web yang digunakan untuk melakukan pelacakan alumni melalui berbagai platform digital seperti LinkedIn, Google Scholar, dan media sosial lainnya.

Sistem ini bertujuan untuk membantu admin kampus dalam:
- Melacak keberadaan alumni
- Mengetahui pekerjaan atau aktivitas alumni
- Mengelola data alumni secara terstruktur

🚀 Fitur Utama

- 🔐 Login Admin
- 📊 Dashboard Alumni
- 🔍 Pencarian Alumni
- 📝 Input Data Alumni
- 🌐 Integrasi link media sosial

🛠️ Teknologi yang Digunakan
- Next.js
- React.js
- Tailwind CSS
- Node.js

📂 Cara Menjalankan Project
```bash
npm install
npm run dev
```
Buka di browser:
```
http://localhost:3000
```
